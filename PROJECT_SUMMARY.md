## 🔧 Customization Guide

### Change Colors
Edit `style.css` root variables:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --accent-color: #f59e0b;
}
```

### Add Games
Edit `games/games-data.js`:
```javascript
{ name: 'Game Name', icon: '🎮', category: 'puzzle', description: 'Description' }
```

### Modify Styling
Edit `style.css` - all CSS is organized with comments:
- Root & Variables
- Global Styles
- Typography
- Buttons
- Navigation
- Components
- Responsive Design

### Add New Fully Playable Game
1. Create game folder: `games/gamename/`
2. Add game logic to `script.js`
3. Add init function: `function initGamename() { }`
4. Add to navigation in `index.html`
5. Add to games-data.js
